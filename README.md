BetterYoutubeUI is a browser extension that significantly improves your YouTube viewing experience on PC by rearranging the layout. It puts the comments exactly where you want them, saving you from endless scrolling.

## ✨ Features

- **Swap Comments and Related Videos**: Moves the Comments section to the right sidebar and Related Videos to the bottom. Read comments without scrolling past the video!
- **Grid View for Related Videos**: Transforms the related videos at the bottom into a clean grid layout (can be toggled via the extension popup).
- **[Optional] Legacy Fullscreen Comments**: A legacy feature kept for users who prefer the extension's original fullscreen comments over YouTube's new official implementation. Can be enabled via the extension popup.
  - Adds a custom "View Comments" button to the fullscreen player controls.
  - Keyboard shortcut **`B`** to quickly toggle the comments window in fullscreen.
  - *Note: As YouTube now officially supports fullscreen comments, this legacy feature may conflict with the default UI and is disabled by default.*
- **Remove Fullscreen Scroll Grid**: Disables the annoying grid view that appears when scrolling down in fullscreen mode.
- **Multilingual Support**: Supports English and Korean based on your browser settings.
