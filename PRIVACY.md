# Privacy Policy

BetterYoutubeUI does not collect, store, transmit, or share any personal information or user data.

## Data Storage

This extension uses Chrome's `storage.sync` API only to save user preferences locally to the user's browser account, including:

* Related videos grid view setting
* Fullscreen comments feature setting
* Comment window state

These settings are not transmitted to any external server and are used solely for providing extension functionality.

## Website Access

The extension only runs on:

* https://www.youtube.com/*

It modifies YouTube page layout elements locally in the browser to improve the viewing experience.

## No Remote Code

This extension does not load or execute remote code. All scripts and resources are packaged within the extension itself.

## No Analytics or Tracking

This extension does not use analytics, tracking tools, advertisements, or telemetry systems.
